@dafcoe/storage-monitor

@dafcoe/storage-monitor is a simple monitoring tool that allows you to check local and session storage usage (total and the used/free space).

When hovered, it expands showing a storage item list sorted by size. Check it out, it's in the bottom right corner.

Customization wise, it allows you to set storage type (localStorage | sessionStorage) and refresh rate (min: 1000ms).
